# ITHAKA — личный сайт Сергея Тимошенко

Портфолио о сайтах, автоматизации и AI-инструментах в двух визуальных режимах:

- `/` — Modern Myth;
- `/archive/` — Archive Terminal.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

GitHub Actions автоматически публикует ветку `main` через GitHub Pages.
